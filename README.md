# 👁️ EyeSense – AI Eye Disease Screening

## 🚀 Overview

EyeSense is an AI-powered healthcare solution that enables early detection of multiple eye-related and systemic conditions using smartphone images.

## 🧠 Problem

Early signs of diseases like anemia, jaundice, and eye infections are often visible in the eye but go undetected due to lack of accessible screening tools.

## 💡 Solution

EyeSense analyzes eye images using computer vision and AI to detect abnormalities and provide early risk assessment.

## 🔍 Features

* Multi-condition detection (Anemia, Jaundice, Infection)
* Image-based analysis
* Eye Health Score
* Early warning alerts

## 🛠️ Tech Stack

* Frontend: React / HTML-CSS-JS
* Backend: Flask (Python)
* AI: OpenCV, Machine Learning

* frontend/ → UI
* backend/ → API + AI logic
